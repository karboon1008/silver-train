Navigation Pattern

This is the most fundamental decision — it shapes every screen in the app. Which navigation style fits best for a gesture-first, card-based travel planner?

A

Tab Bar + Card Feed Classic

Persistent bottom tab bar. Each tab shows a vertically scrollable card feed. Familiar, low learning curve. Microinteractions on card interactions.

B

Gesture-Driven Full-Screen Immersive

No persistent tab bar. Swipe left/right to navigate between sections. Swipe up/down within each section. Cards fill the entire screen for maximum impact.

Pros

Cons

  • Discoverability risk
C

Hybrid: Tab Bar + Bottom Sheet Recommended

Tab bar for top-level navigation with an expandable bottom sheet for contextual quick actions. Cards in the feed; swipe gestures within each section.
