Flow Notes
1
FAB sheet — "Build manually" vs "Generate with AI". Backdrop dims the trip list behind. Sheet springs up from bottom with a drag handle.
2
Manual path — Destination, trip name, date range, traveller count. Minimal form. Creates an empty trip shell; user builds itinerary from Trip Detail.
3
AI preferences — Destination, duration, travel style chips (Calm, Culture, Food, Nature…), and pace (Relaxed / Balanced / Packed). Multi-select chips, no dropdowns.
4
Generating state — Days appear staggered as the OpenAI response streams in. Day 1 loads first, shimmer skeletons hold space for later days. Sub-label shows a calm status message.
5
Review & refine — Full itinerary shown. User can swipe stops to reorder, tap to edit, or use the quiet ✦ "Suggest more" button per day. "Save Trip" commits to Supabase.
6
Explore screen — Category filter strip, AI Generate banner, destination cards with image, name, blurb, and a type tag. Tapping a card → Destination detail with option to add to a trip.
